Job Portal Project
OverviewThis is a web-based job portal application built using Java, Spring Boot, and Thymeleaf. The application allows users to create accounts, post job openings, and apply for jobs.

Features- User registration and login
- Job posting and management
- Job application and tracking
- Admin dashboard for managing users and jobs

Technologies Used- Java 11
- Spring Boot 2.7
- Thymeleaf 3.1
- MySQL 8.0
- Maven 3.8

Project Structure- com.jobportal: Main package for the application
- com.jobportal.config: Configuration classes for the application
- com.jobportal.controller: Controller classes for handling requests
- com.jobportal.model: Entity classes for the database
- com.jobportal.repository: Repository classes for database operations
- com.jobportal.service: Service classes for business logic

Getting Started1. Clone the repository: git clone https://github.com/Mssmuniraju/job_portal.git
2. Navigate to the project directory: cd job_portal
3. Build the project: mvn clean install
4. Run the application: mvn spring-boot:run
5. Access the application: http://localhost:8080
